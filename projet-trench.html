Trench Hero

Game Design Phaser & Tiled

Trench Hero est un projet de jeu vidéo sur lequel j'ai travaillé, pensé et conçu pour être adapté à une borne d'arcade classique.

Le développement s'est appuyé sur le framework JavaScript Phaser, particulièrement adapté pour la création de jeux web en 2D interactifs et performants.

Pour la conception des niveaux et des cartes, nous avons utilisé Tiled (un éditeur de cartes par tuiles), ce qui nous a permis de structurer les décors, les grilles de déplacement, et l'univers du jeu avec une grande précision.
